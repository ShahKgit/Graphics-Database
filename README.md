# Graphics Database Project

## Overview
A Python application built using Pygame that manages a database of soccer players and related statistics. The system allows users to add, modify, delete, and view player entries through a graphical interface.

## Features
- Add, edit, and delete player records
- Storage using file I/O
- Graphical user interface built with Pygame
- Interactive screens, buttons, and typed user input visible on the interface
- Input validation to ensure entries are allowed
- Display of stored database entries

## Implementation Details
The project applies Python programming fundamentals along with file handling. Pygame was used to create a structured interface with interactive components.

Data is stored locally using file I/O.

## Technologies Used
- Python
- Pygame
- File I/O

## Purpose
This project demonstrates the application of programming logic and graphics development to build a functional database system with a user-friendly interface.
